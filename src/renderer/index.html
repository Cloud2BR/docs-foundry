Docs Foundry

A powerful local-first desktop editor for Markdown documentation with multi-tab editing, workspace search, live preview, document outline, and more.

Write

Edit Markdown in a multi-tab split-pane workspace with auto-save and keyboard shortcuts.

Find

Command palette, workspace-wide search, in-file find & replace, and quick file open.

Navigate

File tree, document outline, breadcrumbs, and zen mode for distraction-free writing.

Preview

Live-rendered headings, lists, tables, code, task lists, strikethrough, footnotes, and more.

Organize

Create, rename, and delete files and folders directly from the sidebar.

Export your rendered documents to standalone HTML or PDF files.
